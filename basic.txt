git clone ---- To clone the repo or download the repo
git status ---- to display the state of the repository and staging area. 
git log ----displays all of the commits in a repository's history.
git branch ----create, list, rename, and delete branches.
git checkout -b ---- navigate between the branches created by git branch 
git commit -m ---captures a snapshot of the staged changes in your project.
git add <filenames> ---- add the changed files q
git remote -v ---- to add, change and delete them.
git tag ----create semantic version number identifier tags that correspond to software release cycles.



* pull ---pull request is a proposal to merge a set of changes from one branch into another.
* push --- used to upload local repository content to a remote repository.
